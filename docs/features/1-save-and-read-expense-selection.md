Saving expense selection
1. On transactions page, when user makes a selection, this selection should be stored in LocalStorage
2. In LocalStorage, key should be combination of transaction year, month, day and amount and value should be selected value

Reading saved selections and applying them
1. When transactions page is loaded, all saved selections in LocalStorage for a user selected year and month should be read and applied
2. When different set of transactions is filtered by year and or month, saved selections for given year and month should be read and applied on page
